Amazon Transcribe is a speech recognition service that makes it easy for developers to add speech-to-text capability to their applications. Transcribe uses advanced deep learning technologies to provide highly accurate speech-to-text transcription. Transcribe is easy to use, and there is no machine learning experience required to start using this service. Transcribe makes it easy for developers to add speech-to-text capability to their applications, and also makes it easy for developers who have experience building machine learning models to customize and train their own models for specific use cases.
